Checklist for Data Manipulation RadaR

Provide users a checklist for quick data manipulation (RadaR).

Solutions

  • Convert data type (SQL, R)
  • Wide format or long format
  • Check and fix if data is out of range
  • Search for nulls using conditional formatting and filters or using SQL to filter
  • Using Remove Duplicates function in excel or DISTINCT IN SQL
"If value is wrong, check if the data is biased or reliable
If misspellings, locate or the misspellings
If extra spaces, remove by using TRIM function"

Warning/Common error

  • Not cheking for spelling errors
  • Forgetting to document errors
  • Not checking for misfield values
  • Overlooking missing values
  • Looking at a subset of data not the whole picture
  • Losing track of the business objectives
  • Not fixing the source of the error
  • Not analyzing the system prior to data cleaning
  • Not backing up prior to data
  • Not accouting for data cleaning in your deadlines/process

Steps

  1. Ensure data
  2. Identify issues in the data: Identify by going through the data/ Visualizing data to detect any irregularities
  3. Define the scope of the cleaning process: Which fields or data points needed to be cleaned and create any plan for how to handle any issues
